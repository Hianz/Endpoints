# Endpoints

## Installation

**CocoaPods:**

```
pod "Endpoints", "~> 0.3"
```

**Carthage:**

```
github "tailoredmedia/Endpoints.git" ~> 0.3
```

**Swift Package Manager:**

```
.Package(url: "https://github.com/tailoredmedia/Endpoints.git", majorVersion: 0, minor: 3),
```

## Platform support

- iOS 8
- tvOS 9
- macOS 10.11
- watchOS 2.0
